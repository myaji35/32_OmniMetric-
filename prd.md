## 1. 프로젝트 개요 (Overview)

### 1.1 제품 비전

OmniMetric(제품명: **XAISimTier** - Explainable AI Simulation Frontier)은 업무 데이터를 입력하면 125종 이상의 AI 알고리즘을 병렬 실행하여 최적의 모델을 자동 선정하고, **분석 리포트 → What-if 시뮬레이션 → 목표 달성 최적화**까지 원클릭으로 제공하는 AI 의사결정 엔진입니다.

### 1.2 핵심 철학

> **"분석은 '예측'이고, 시뮬레이션은 '의사결정'이다."**
> AI는 결과를 맞추는 도구가 아니라, 더 나은 선택을 미리 보여주는 도구입니다.

> **"교육은 시작점이 아니다. 결과가 시작점이다."**
> 통계/AI 지식 없이도 데이터로 즉시 의사결정이 가능한 UX를 추구합니다.

### 1.3 핵심 미션

- **통찰의 자동화**: 데이터 과학자 없이도 데이터 내 핵심 동인(Key Driver)을 즉시 판별
- **신뢰의 수치화**: $R^2$ 기반의 최적 모델 선정으로 의사결정의 객관적 근거 확보
- **설명 가능한 AI (XAI)**: "왜?"라는 질문에 통계적 근거와 자연어 리포트로 응답
- **의사결정 시뮬레이션**: What-if 시나리오를 가상 실행하여 최적의 선택을 도출
- **목표 달성 최적화**: 예산, ROI 등 제약조건을 고려한 최적 변수(X) 조합 탐색

## 2. 4대 AI 분석 유형 (Core Analysis Types)

### 2.1 회귀분석 (Regression) - 숫자로 결과를 예측

- **목적**: 매출액, 점수, 온도 등 연속형 수치를 예측
- **알고리즘**: 30종 지원
- **활용 예시**:
  - "이 자리에서 창업하면 예상 매출은 얼마일까?"
  - "광고비를 얼마 올리고 가격을 얼마 써야 매출이 최대가 될까?"
  - "어떻게 하면 생산량은 18% 늘리고 원가는 11% 줄일 수 있을까?"
- **시뮬레이션 지원**: 모델식 → Sim(Y예측, X최적화) 지원

### 2.2 분류분석 (Classification) - O/X 문제를 풀어요

- **목적**: 합격/불합격, 정상/불량 등 이진 분류 예측
- **알고리즘**: 17종 지원
- **활용 예시**:
  - "이 입지는 성공할까, 실패할까?"
  - "이 설비는 고장이 발생할 것인가, 아닌가?"
  - "이 고객은 우리 제품을 구매할까, 안 할까?"
- **시뮬레이션 지원**: 모델식 → Sim(Y예측, X최적화) 지원

### 2.3 다중분류분석 (Multi-class Classification) - 여러 선택지 중 하나를 골라요

- **목적**: 등급/카테고리 등 3개 이상의 클래스로 분류
- **알고리즘**: 17종 지원
- **활용 예시**:
  - "이 상권의 잠재력은 어느 등급일까? (활성화/성장/정체)"
  - "이 고객은 VIP, 우수, 일반 중 어디에 속할까?"
  - "설비 상태는 정상/주의/위험 중 어느 상태인가?"
- **시뮬레이션 지원**: 모델식 → Sim(Y예측, X최적화) 지원

### 2.4 시계열분석 (Time-Series) - 시간의 흐름을 예측해요

- **목적**: 시간 흐름에 따른 미래 물동량, 수요, 추세 예측
- **알고리즘**: 61종 지원 (시계열변수 필드 약 500개 자동생성)
- **활용 예시**:
  - "이 입지의 향후 1년간 매출 흐름은 어떨까?"
  - "다음 주 비 오는 주말에 주문이 얼마나 들어올까?"
  - "다음 분기 우리 제품의 수요는 어떻게 변할까?"
- **시뮬레이션 지원**: 모델식 → Sim(Y예측, X최적화) 지원

## 3. 주요 기능 요구사항 (Core Features)

### 3.1 알고리즘 토너먼트 엔진 (125+ Algorithms)

| 분석 유형 | 알고리즘 수 | 지원 기능 |
|---------|---------|---------|
| 회귀분석 | 30종 | 모델식 → Sim(Y예측, X최적화) |
| 판별분석 | 17종 | 모델식 → Sim(Y예측, X최적화) |
| 다중분류분석 | 17종 | 모델식 → Sim(Y예측, X최적화) |
| 시계열분석 | 61종 | 모델식 → Sim(Y예측, X최적화) + 자동 파생변수 500개 |
| **합계** | **125종+** | |

- **전수 조사**: 모든 알고리즘을 입력 데이터에 무차별 대입(Brute-force)
- **승자 독식**: 각 모델의 $R^2$, Adj. $R^2$, P-value를 비교하여 최적 모델 자동 선정

### 3.2 초고속 병렬 처리 시스템 (Parallel Engine)

- **멀티프로세싱 최적화**: 125여 개의 알고리즘을 각각 독립 워커(Worker)에 할당하여 동시 실행
- **API 비동기 응답**: 대규모 데이터셋 분석 시 작업 ID를 즉시 반환하고, 완료 시 Webhook 또는 폴링으로 결과 전달

### 3.3 XAISimTier 5단계 분석 파이프라인

```
1. 사용자 설정     → 분석 목표(Y)와 원인 변수(X), 분석 방법 선택
       ↓
2. 자동 리포팅     → 변수별 분포, 변수 간 상관관계 등 AI 자동 리포트 생성
       ↓
3. 모델링 & 검증   → 디시전트리 등 AI 모델로 핵심 패턴 시각화, 성능 자동 검증
       ↓
4. What-if 시뮬레이션 → 원인 변수(X)를 조정했을 때 목표(Y)가 어떻게 변할지 시나리오 탐색
       ↓
5. 최적화          → 설정된 목표(Y) 달성을 위한 최적의 원인 변수(X) 조합을
                     예산, ROI 등 제약조건을 고려하여 탐색
```

### 3.4 WhatDataAI 분석 (데이터 사전 분석)

- AI 활용 가능한 분석 방법과 데이터를 자동 선별
- 입력 데이터의 특성에 따라 적합한 분석 유형 자동 추천

### 3.5 현황분석 (데이터 탐색)

- 다양한 차트 분석 지원
- 워드클라우드 분석 지원
- 주제도 분석 (위치, 히트맵 등 제작 지원)

### 3.6 수학적 공식 및 변수 기여도 도출

- **공식 변환**: 선정된 모델을 수학 공식($Y = w_1X_1 + w_2X_2 + ... + b$)으로 출력
- **영향력 분석**: 각 변수가 결과값에 미치는 가중치를 백분율 및 방향성(+, -)으로 수치화
- **디시전트리 분석**: XAI 변수 픽처링을 통한 핵심 패턴 시각화

### 3.7 Sim4Brief Report (실시간 자동 리포팅)

- **자연어 생성 (NLG)**: 통계적 수치를 "A 변수가 10% 증가할 때 B가 5% 상승함"과 같은 인간의 언어로 변환
- **사용자 정의 분석**: 회귀, 판별, 다중분류, 시계열 분석별 파이프라인 기반 AI 분석 결과 실시간 자동 리포팅
- **AI Q&A**: 분석 결과를 기반으로 AI에게 자유롭게 질문 가능

### 3.8 행동 시나리오 변환 (Action Scenario Converter)

분석 결과를 즉시 실행 가능한 업무 규칙(Action Rule)으로 자동 변환하는 기능입니다.

- **규칙 자동 생성**: 분석 결과를 "IF 조건 THEN 행동" 형태의 실행 가능한 규칙으로 변환
  - 예: "위험도 > 70% → 현장 점검 2회 증가"
  - 예: "고객 이탈 확률 > 60% → 무료 사이드메뉴 쿠폰 발송"
  - 예: "설비 고장 확률 > 80% → 정비 주기 단축 알림"
- **임계값 기반 트리거**: 사용자가 설정한 임계값에 따라 자동으로 행동 시나리오를 분기
- **우선순위 자동 산정**: 영향도(Impact) x 발생확률(Probability) 기반으로 행동 시나리오의 우선순위를 자동 배정
- **연동 액션**: Webhook, 이메일, 슬랙 등 외부 시스템에 행동 시나리오를 자동 전달하여 실무자가 즉시 실행 가능
- **시나리오 이력 관리**: 생성된 행동 시나리오와 실행 결과를 추적하여 효과 측정

> 핵심 원칙: 분석의 최종 결과물은 '차트'나 '숫자'가 아니라, **즉시 실행 가능한 '행동 시나리오'**여야 한다.

### 3.9 자가 보정 MLOps (Auto-Retraining)

- **오차 감지**: 실제 운영 데이터와 예측 공식의 오차가 설정 임계값(Threshold)을 벗어날 경우 자동 감지
- **재학습 파이프라인**: 오차 발생 시 125개 알고리즘 토너먼트를 재개최하여 공식을 실시간 보정

## 4. LLM + XAISimTier 시너지 아키텍처

### 4.1 역할 분담

| 구분 | LLM (GPT 기반 지식그래프) | XAISimTier |
|------|----------------------|------------|
| **역할** | "무엇이 있었나?" (과거 정보 검색) | "무엇을 해야 하나?" (미래 예측 및 처방) |
| **핵심 기능** | 지식의 자동화 | 결정의 자동화 |
| **데이터** | 규정, 문서, 업무 지식 등 축적된 데이터 | KPI 목표 달성을 위한 변수 최적화 |

### 4.2 시너지 플로우

```
1. LLM이 '맥락'을 제공
   → 지식그래프가 분석에 필요한 업무 규칙, 변수 정의, 제약조건을 제공

2. XAISimTier가 '최적해'를 계산
   → 주어진 제약조건 하에서 목표 달성을 위한 최적의 조합을 시뮬레이션

3. '근거 기반 리포트'로 완성
   → 지식(Why)과 예측(What-if)이 결합된 완전한 의사결정이 가능
   → 비용, ROI, 제약조건을 고려한 전략적 의사결정 모델을 자동 생성
```

## 6. 기술 아키텍처 및 스택 (Technical Stack)

- **Backend**: Python 3.11 (FastAPI) - 비동기 I/O 및 고성능 API 처리
- **Parallelism**: Ray 또는 Celery (Redis 기반) - 125+ 알고리즘 병렬 분산 처리
- **Analysis**: PyCaret, Scikit-learn, Statsmodels - 통계 및 ML 라이브러리
- **XAI**: SHAP, LIME - 모델 해석 엔진
- **Time-Series**: Prophet, ARIMA, 자동 파생변수 생성 엔진
- **LLM Integration**: GPT 기반 지식그래프 연동 (맥락 제공 + 자연어 리포트)
- **Interface**: RESTful API (JSON 포맷 응답)

## 7. 데이터 플로우 (Data Flow)

```
입력 (CSV/JSON)
  ↓
WhatDataAI 분석 (데이터 자동 선별 + 분석 방법 추천)
  ↓
현황분석 (차트, 워드클라우드, 주제도)
  ↓
분산 (125개 알고리즘 워커 병렬 실행)
  ↓
경쟁 (R² 계산 + 모델 성능 비교)
  ↓
선정 (최적 모델 확정)
  ↓
XAI 해석 (디시전트리 + SHAP/LIME)
  ↓
What-if 시뮬레이션 (시나리오 탐색)
  ↓
최적화 (제약조건 하 최적 변수 조합 도출)
  ↓
출력 (공식 + 자연어 리포트 + Sim4Brief Report)
  ↓
행동 시나리오 변환 (IF-THEN 규칙 자동 생성 + 외부 시스템 전달)
  ↓
감시 (자가 보정 MLOps)
```

## 10. 실무자 접근 전략 (User Acquisition)

### 10.1 실무자 자연 성장 시나리오

통계/AI 지식이 없는 실무자도 즉시 활용 가능하도록 3단계 접근:

1. **1단계 (15분)**: 데이터를 넣으면 즉시 '완주' 경험 → 시각적 결과를 바로 보는 첫 경험
2. **2단계 (업무 반복)**: 회의, 보고서에 결과를 쓰다 보면 "왜?"라는 자연스러운 궁금증이 생김
3. **3단계 (필요할 때만)**: 궁금증이 생겼을 때 '선택' 해서 진화 → 짧은 설명을 원할 때 클릭

### 10.2 5가지 데이터 활용법 프레임워크

실무자가 데이터를 활용할 때 반드시 5가지 중 하나에 해당:

1. **현상파악**: 무슨 일이 일어나고 있는가?
2. **원인분석**: 왜 일어났는가?
3. **예측**: 앞으로 어떻게 될 것인가?
4. **비교**: 어떤 것이 더 나은가?
5. **시뮬레이션**: 조건을 바꾸면 결과가 어떻게 달라지는가?
